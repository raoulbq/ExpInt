class expint.methods.Method.Method(rhs=None)¶

Interface for integration schemes.

Parameters:rhs – Right-hand side to solve.
integrate(x0, t0, tend, N)¶

This function should implement the time integration for t from t0 to tend, given the initial condition x(t0) = x0. N is the number of timesteps.

Convenience function to set right-hand side to rhs

Marker class for adaptive methods. Problem class checks if method is derived from this to determine function signature.

class expint.methods.Method.ExponentialMethod(rhs=None, matexp='cpade')¶

Defines an integration method that requires a matrix exponential.

Parameters:
  • rhs – instance of a DfRHS (fRHS not sufficient, need Jacobian)
  • matexp – describes type of matrix exponential to use
Notes:
  • Certain matrix exponential algorithms require the actual matrix. The Df argument of the DfRHS class constructor must be a numpy array; a FunctionType will not work!
  • If a method requires a specific matrix exponential, it should overwrite the __init__ function and explicitly set the variable self.matexp there.
matexp(A, v=None, h=1.0)¶
Parameters:
  • A – matrix to take exponential of
  • v – vector to multiply exp(h*A) with (if given)
  • h – scaling factor in exp(h*A)
Returns:

exp(h*A) if v is not given.
Returns:

exp(h*A)*v is v is given.
