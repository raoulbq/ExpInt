Exp4¶

The Exp4 module contains two classes: the basic Exp4 scheme and an adaptive version called Exp4_adaptive.

Exp4.Exp4¶

class expint.methods.Exp4(rhs, abstol=1e-06, reltol=1e-06, stats=False)¶

Exp4 exponential integration algorithm as described by Hochbruck, Lubich and Selhofer (1998)

Parameters:
  • rhs – instance of class implementing RHS interface
  • stats – True means to keep track of statistics
Notes:
  • This algorithm is optimized for large, sparse systems. Due to the construction of Krylov subspaces, its performance is bad for small-dimensional systems.

Exp4.Exp4_adaptive¶

class expint.methods.Exp4_adaptive(rhs, abstol=1e-06, reltol=1e-06, stats=True, krylovargs=None)¶

Exp4 exponential integration algorithm as described by Hochbruck, Lubich and Selhofer (1998)

Parameters:
  • rhs – instance of class implementing RHS interface
  • abstol – absolute tolerance (default 1e-6)
  • reltol – relative tolerance
  • stats – True means to keep track of statistics
  • krylovargs – arguments given to constructor of KrylovPhi class
Notes:
  • This algorithm is optimized for large, sparse systems. Due to the construction of Krylov subspaces, its performance is bad for small-dimensional systems.
integrate(x0, t0, tend, h=None, abstol=None, reltol=None)¶
Parameters:
  • x0 – initial value
  • t0 – initial time
  • tend – end time
  • h – initial timestep size. If not specified, (tend-t0)/100 will be used.
  • abstol – absolute tolerance. Overwrites self.abstol
  • reltol – relative tolerance. Overwrites self.reltol
Returns:

tuple (t,y)
